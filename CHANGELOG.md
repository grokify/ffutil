## [v0.1.0] - 2026-02-21

### Highlights

- Go wrapper for FFmpeg and FFprobe with type-safe command building, media probing, and hardware encoder detection

### Added

- Type-safe command builder with fluent API for constructing FFmpeg commands ([`3446971`](https://github.com/grokify/ffutil/commit/3446971))
- `Probe()` function returning detailed media info (duration, resolution, codecs, streams) ([`3446971`](https://github.com/grokify/ffutil/commit/3446971))
- `Duration()`, `Resolution()`, `HasAudio()`, `HasVideo()` convenience functions ([`3446971`](https://github.com/grokify/ffutil/commit/3446971))
- Hardware encoder detection with `BestH264Encoder()` and `BestHEVCEncoder()` ([`3446971`](https://github.com/grokify/ffutil/commit/3446971))
- Cross-platform support for VideoToolbox (macOS), NVENC (NVIDIA), QuickSync (Intel), AMF (AMD), VA-API (Linux) ([`3446971`](https://github.com/grokify/ffutil/commit/3446971))
- `Version()`, `ProbeVersion()`, `Available()` for checking FFmpeg/FFprobe installation ([`3446971`](https://github.com/grokify/ffutil/commit/3446971))
- `ListEncoders()` function to enumerate all available video encoders ([`3446971`](https://github.com/grokify/ffutil/commit/3446971))
